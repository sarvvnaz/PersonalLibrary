
# Personal Library

Personal Library is a simple **web-based personal library application** built with React.

It allows users to manage a small collection of books directly in the browser through an interactive user interface.

---

## What the project does

The application allows users to:

* Add books by providing:

  * title
  * author
  * publication year
  * cover image URL
* View all added books in a navigation sidebar
* Search books by title or author
* Select a book to view its details
* Remove a book from the library

## How to run the project locally

### 1. Install dependencies

```bash
npm install
```

### 2. Start the development server

```bash
npm start
```

The application will be available at:

```
http://localhost:3000
```

---


## Build for production

To create a production build:

```bash
npm run build
```
